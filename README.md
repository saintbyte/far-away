# Far Away

Это банальный клон telegra.ph но с поддержкой темной темы. Темную тему врубать не потребуется она берется прям из настроек ОС/Браузера. Оптимизированно для запуска на vercel.com. 

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fsaintbyte%2Ffar-away)

## Установка на vercel
1. Создайте на vercel базу Postgres или еще где-то.
1.1 Если вы создавали базу не на vercal то вам понадобиться поместить database url в переменную среды POSTGRES_URL
2. Соделайте форк репы far away
3. Деплойте на vercel из вашей репы.
